# 客户信息系统
## 主要功能
1. 添加客户
2. 修改客户
3. 删除客户
4. 客户列表
5. 退出

## 操作方式
- 用户输入数字选择功能(1~5)
    - 输入1，进入“添加客户”界面，需要填写姓名、性别、年龄、电话、邮箱这些信息
    - 输入2，进入“修改客户”，首先要输入要修改客户的编号，然后填写信息
    - 输入3，进入“删除客户”，输入要删除客户的编号，再确认是否删除
    - 输入4，进入“客户列表”，显示客户列表信息
    - 输出5，经过确认后退出系统

## 实现技术栈
- C语言:结构体 链表 计划支持文件读写

## 重要总结
1. 链表的头指针至关重要,必须保存好,在函数中最好使用temp来代替phead遍历
2. 接受输入时,对于%d的需要添加检测保护,低于不合法的输入必须清除缓存区
3. 另外为了提升体验,使用getchar()来反馈用户效果很好
4. 为了更广泛的使用,在退出程序时需要释放链表的内存空间
5. 为了提升拓展性,node节点应该通过List结构体内的Node*指针进行使用
